Movie-API

This web-app provides the users with information and details about different movies. Moreover, users can personalize their movie list and create a list of their favorite movies, if they have already signed up themselves and established a user account.

Features

This web-app can:

  • Return a list of ALL movies to the user
  • Return data (description, genre, director, image URL, whether it is featured or not) about a single movie by title to the user
  • Return data about a genre (description) by name
  • Return data about a director (bio, birth year, death year) by name
  • Allow new users to register
  • Allow users to update their user info
  • Allow existing users to deregister.
  • Allow users to add a movie to their list of favorites
  • Allow users to remove a movie from their list of favorites

RESTful API

Useful information about the URL endpoints and request methods.
Request URL HTTP Method Request body data format Response body data format
Get a list of all movies /movies GET None A JSON object holding data about all movies
Get information about a specific movie
(by "movie's title")		 /movies/:title GET None A JSON object holding data about the movie
Get information about a genre
(by "genre's name")		 /movies/genre/:name GET None A JSON object holding all data about the genre

{
"name": "Movie 1 Genre",
"description": "Movie 1 Genre Description"
}

Get information about a director
(by "director's name")		 /movies/directors/:name GET None A JSON object holding all data about the director

{
"name": "Director 1",,
"bio": "Bio Director 1",
"birthYear": 1890,
"deathYear": 2002
}

Register a new user /users POST A JSON object holding data about user's name

{
"name": "user name"
}

A JSON object holding data about the new user

{
"name": "user name",
"id": "6b0378d6-21d6-478f-a299-f5c98ee2fb40"
}

Update user info
(by "user's id")		 /users/:id PUT A JSON object holding new data about the user

{
"name": "user's new name"
}

A JSON object holding data about the updated user

{
"name": "user name",
"id": "6b0378d6-21d6-478f-a299-f5c98ee2fb40",
"favoriteMovies": []
}

Deregister a user
(by "user's id")		 /users/:id DELETE None A message including the info about the deregistered user
Add a movie to user's list of favorites
(by "movie title")		 /users/:userId/:movieTitle POST None A message including the info about the added movie to the list of favorites
Remove a movie from user's list of favorites
(by "movie title")		 /users/:userId/:movieTitle DELETE None A message including the info about the removed movie from the list of favorites
