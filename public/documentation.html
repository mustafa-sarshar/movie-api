Movie-API

This web-app provides the users with information and details about different movies. Moreover, users can personalize their movie list and create a list of their favorite movies, if they have already signed up themselves and established a user account.

Features

This web-app can:

  • Return a list of ALL movies to the user
  • Return data (description, genre, director, image URL, whether it is featured or not) about a single movie by title to the user
  • Return data about a genre (description) by name
  • Return data about a director (bio, birth date, death date) by name
  • Return data about an actor/actress (bio, birth date, death date) by name
  • Allow users to login
  • Allow new users to register
  • Allow users to update their user info
  • Allow existing users to deregister.
  • Allow users to add a movie to their list of favorites
  • Allow users to remove a movie from their list of favorites
